CSS Selectors Overview

Learn how to target HTML elements using ID, Class, Tag, Group & more.

1. Universal Selector (*)

Selects all elements on the page. Useful for resets or base styling.

  • Applies styles globally
  • Often used for CSS reset
  • Affects all elements - use carefully
  • Great for setting box-sizing for entire page
* { margin:0; padding:0; box-sizing:border-box; }

2. Type Selector (Element Selector)

Selects elements by tag name such as h1, p, div, etc.

  • Applies style to all tags of a type
  • Ideal for typography
  • Lower specificity than class or ID
  • Good for global tag formatting
p { font-size:16px; color:#333; }

3. Class Selector (.)

Targets an element by its class attribute. Reusable many times.

  • Most commonly used selector
  • Perfect for reusable UI components
  • Elements can have multiple classes
  • Moderate specificity level
.title { font-weight:bold; color:green; }

4. ID Selector (#)

Targets a unique element by its id. Should be used only once per page.

  • Highest specificity (stronger than class)
  • Used for unique layout blocks
  • Not reusable like class selectors
  • Helps in JavaScript element selection
#mainHeader { background:#eee; padding:10px; }

5. Group Selector ( , )

Used to apply the same styling to multiple selectors at once.

  • Reduces duplicate CSS code
  • Makes stylesheet cleaner
  • Good for styling headings or shared elements
  • Improves maintainability
h1, h2, h3 { font-family: Arial, sans-serif; }
